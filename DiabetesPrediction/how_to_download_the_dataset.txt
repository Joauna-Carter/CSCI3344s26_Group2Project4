The dataset for this project is hosted by Kaggle. To download the necessary dataset for this project, please follow the instructions below.

1. Go to https://www.kaggle.com/uciml/pima-indians-diabetes-database

2. Click on the ‘Download’ button

3. Kaggle will prompt you to sign in or to register. If you do not have a Kaggle account, you can register for one using any email.

4. After the download is complete, unzip the zip file and move the file ‘diabetes.csv’ into your project folder.
